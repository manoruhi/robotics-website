# STEMCITY® – React Website

React project inspired by [stemcity.in](https://stemcity.in/) – “Changing the way of learning.”

## Navbar

- **Home** – Hero / landing
- **Projects** – Learning engine & projects
- **Courses** – Course listings
- **Vlog** – Blog / updates
- **Shop** – Shop by category
- **Search bar** – Search input with submit
- **Cart** – Cart icon with count

## Footer

- Brand (STEMCITY®), tagline, social links
- Quick Links, Products, Contact
- Copyright and legal links (Privacy, Terms, Cookie Policy)

## Run the project

**Option 1 – Double-click (Windows)**  
Run `run.bat`, then open **http://localhost:5173** in your browser.

**Option 2 – Terminal**

```bash
npm install
npm run dev
```

Then open **http://localhost:5173**.

## Build

```bash
npm run build
npm run preview
```

## Tech

- React 18 + Vite 5
